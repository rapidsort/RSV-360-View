RSV 360 View is a jQuery Rotator.

== Description ==

RSV 360 View plugin for displaying Image as 360 Degree View. It uses a media image for 360 view. you can fill your pages with 360 Rotators. 

This Plugin is based on <a href="https://cdnjs.com/libraries/jquery-mousewheel" target="_blank">jQuery Mousewhee</a>.

== Installation ==

1. Upload the <strong>rsv-360view</strong> folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode <strong>[rsv360_view image="image.jpg" width="900" height="300" stitched="4320" responsive="true"]</strong> in the content area of a page or post where you want the 360 Rotator to appear.


== Frequently Asked Questions ==

= How do I insert the RSV 360 View? =
You can insert the RSV 360 View by pasting the shortcode `[rsv360_view image="image.jpg" width="900" height="300" stitched="4320" responsive="true"]` into the content area of a page or post.  **Be sure to use the HTML editor when inserting shortcodes!** To customize your 360 Rotator, there are optional attributes that can be used with the shortcode.
